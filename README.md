### Download and explore 3D EM volumes

This repo contains some simple utility scripts to download and play with HHMI Janelia's [CellMap](https://open.quiltdata.com/b/janelia-cosem-datasets/tree/) (3D FIB-SEM volumes with various annotations for some of the volumes) and [OpenOrganelle](https://openorganelle.janelia.org/) datasets. Work in progress.
